# Gilchrist Module 2 Assignment

## Project Overview
This project is a simple faculty web presence prototype created as part of a web development assignment focused on interface design, HTML structure, and professional workflow practices.

The goal of this project is to design and implement a basic faculty website that allows students to view faculty information and submit contact inquiries. The emphasis of the project is on design thinking, layout structure, and development process rather than advanced functionality or visual polish.

-------------------------------------------------------------------------------------------------------

## Purpose of the Site
Students often need quick access to faculty information such as courses taught, office hours, and contact methods. This prototype demonstrates a clear and organized interface that improves accessibility to this information through a simple web design.

The website includes:
- A faculty profile homepage
- A contact form for gathering user information

-------------------------------------------------------------------------------------------------------

- **Bootstrap 5** – Responsive layout and basic styling
